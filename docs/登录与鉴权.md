# 认证系统接入指南

## 概述

本项目包含两套独立的认证系统：
- **Admin 后台认证** - 供后台管理人员使用
- **App 用户认证** - 供普通用户使用

---

## 一、Admin 后台认证

### 基础信息

| 项目 | 值 |
|------|-----|
| 接口前缀 | `/auth` |
| Token 前缀 | `Bearer` |
| Token 有效期 | 配置决定 |

### 接口列表

#### 1. 获取验证码

```
GET /auth/code
```

**响应示例：**
```json
{
  "img": "data:image/png;base64,...",
  "uuid": "code-xxx-xxx"
}
```

#### 2. 获取 RSA 公钥

```
GET /auth/public-key
```

**响应示例：**
```json
{
  "publicKey": "MIGfMA0GCSqGSIb3DQEBAQUAA4GNADCBiQKBgQC..."
}
```

#### 3. 登录

```
POST /auth/login
Content-Type: application/json
```

**请求体：**
```json
{
  "username": "admin",
  "password": "RSA加密后的密码",
  "code": "1234",
  "uuid": "code-xxx-xxx"
}
```

**响应示例：**
```json
{
  "token": "Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "id": 1,
    "username": "admin",
    "nickName": "管理员",
    "avatar": "https://...",
    "email": "admin@example.com",
    "enabled": true,
    "authorities": [
      {"authority": "ROLE_ADMIN"}
    ]
  }
}
```

#### 4. 获取当前用户信息

```
GET /auth/info
Authorization: Bearer {token}
```

#### 5. 退出登录

```
DELETE /auth/logout
Authorization: Bearer {token}
```

## 三、前端接入流程

### Admin 后台接入

```javascript
// 1. 获取验证码
const codeResponse = await fetch('/auth/code');
const { img, uuid } = await codeResponse.json();

// 2. 显示验证码图片给用户
// ...

// 3. 获取公钥
const keyResponse = await fetch('/auth/public-key');
const { publicKey } = await keyResponse.json();

// 4. 用户输入后，使用 RSA 公钥加密密码
const encryptedPassword = rsaEncrypt(password, publicKey);

// 5. 登录
const loginResponse = await fetch('/auth/login', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    username,
    password: encryptedPassword,
    code,
    uuid
  })
});

const { token, user } = await loginResponse.json();

// 6. 保存 token
localStorage.setItem('adminToken', token);

// 7. 后续请求带上 token
fetch('/api/some-endpoint', {
  headers: { 'Authorization': token }
});
```

### App 用户接入

```javascript
// 工具函数
async function getPublicKey() {
  const response = await fetch('/api/app/auth/public-key');
  const { publicKey } = await response.json();
  return publicKey;
}

async function saveTokens(data) {
  localStorage.setItem('accessToken', data.accessToken);
  localStorage.setItem('refreshToken', data.refreshToken);
  localStorage.setItem('user', JSON.stringify(data.user));
}

function getDeviceId() {
  let deviceId = localStorage.getItem('deviceId');
  if (!deviceId) {
    deviceId = generateUUID();
    localStorage.setItem('deviceId', deviceId);
  }
  return deviceId;
}

// 注册
async function register(email, password, nickname, deviceName) {
  const publicKey = await getPublicKey();
  const encryptedPassword = rsaEncrypt(password, publicKey);
  const deviceId = getDeviceId();

  const response = await fetch('/api/app/auth/register', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      email,
      password: encryptedPassword,
      nickname,
      deviceId,
      deviceName
    })
  });

  const data = await response.json();
  saveTokens(data);
  return data;
}

// 登录
async function login(email, password, deviceName) {
  const publicKey = await getPublicKey();
  const encryptedPassword = rsaEncrypt(password, publicKey);
  const deviceId = getDeviceId();

  const response = await fetch('/api/app/auth/login', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      email,
      password: encryptedPassword,
      deviceId,
      deviceName
    })
  });

  const data = await response.json();
  saveTokens(data);
  return data;
}

// 刷新 Token
async function refreshToken() {
  const refreshToken = localStorage.getItem('refreshToken');
  if (!refreshToken) {
    throw new Error('No refresh token');
  }

  const response = await fetch('/api/app/auth/refresh', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ refreshToken })
  });

  const data = await response.json();
  saveTokens(data);
  return data;
}

// 退出登录
async function logout() {
  const accessToken = localStorage.getItem('accessToken');
  const deviceId = getDeviceId();

  await fetch(`/api/app/auth/logout?deviceId=${deviceId}`, {
    method: 'POST',
    headers: { 'Authorization': `Bearer ${accessToken}` }
  });

  localStorage.removeItem('accessToken');
  localStorage.removeItem('refreshToken');
  localStorage.removeItem('user');
}

// 请求拦截器示例
// 在 Access Token 过期前自动刷新
async function apiRequest(url, options = {}) {
  let accessToken = localStorage.getItem('accessToken');

  // 检查 token 是否即将过期（提前 1 小时）
  if (isTokenAboutToExpire(accessToken)) {
    try {
      const data = await refreshToken();
      accessToken = data.accessToken;
    } catch (e) {
      // 刷新失败，跳转到登录页
      logout();
      window.location.href = '/login';
      return;
    }
  }

  const response = await fetch(url, {
    ...options,
    headers: {
      ...options.headers,
      'Authorization': `Bearer ${accessToken}`
    }
  });

  return response;
}
```

## 四、RSA 加密示例

### 使用 JSEncrypt 库

```javascript
import JSEncrypt from 'jsencrypt';

function rsaEncrypt(text, publicKey) {
  const encrypt = new JSEncrypt();
  encrypt.setPublicKey(publicKey);
  return encrypt.encrypt(text);
}
```

### 使用 Node.js crypto 模块

```javascript
import crypto from 'crypto';

function rsaEncrypt(text, publicKey) {
  return crypto.publicEncrypt({
    key: publicKey,
    padding: crypto.constants.RSA_PKCS1_PADDING
  }, Buffer.from(text)).toString('base64');
}
```

## 五、错误码说明

| HTTP 状态码 | 说明 |
|------------|------|
| 400 | 请求参数错误 |
| 401 | 未授权或 Token 过期 |
| 403 | 无权限访问 |
| 500 | 服务器内部错误 |

## 六、注意事项

1. **设备 ID 生成**：每个设备应生成唯一的 deviceId 并持久化存储
2. **Token 刷新时机**：建议在 Access Token 过期前 1 小时内进行刷新
3. **密码加密**：密码必须使用 RSA 公钥加密后传输
4. **双系统分离**：Admin 和 App 认证完全独立，不要混用接口
5. **多设备登录**：App 支持多设备同时登录，每个设备有独立的 Refresh Token
