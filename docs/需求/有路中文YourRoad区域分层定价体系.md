# 有路中文 YourRoad 区域分层定价体系

有路中文 YourRoad 是面向国际中文教育的多模态教学资源平台。平台以词汇、语法、汉字、文化、话题及"中文+"专业中文等核心学习要素为基础，提供结构化、多模态、可关联、可持续更新的中文学习与教学资源。

本定价主要参考以下因素：中文学习资源的专业性、完整性与权威性；词汇、语法、汉字、文化、话题等资源之间的关联程度；图文、音频、视频、动画、课件及互动资源等多模态形态；机构、学校及教育组织的成员授权、权限管理、使用统计与数据报告；教材配套、专业中文、系统对接、私有化部署等项目服务。其核心逻辑是个人版面向完整资源订阅，机构版面向批量资源授权，Enterprise / 定制项目面向项目制解决方案。

## 一、区域价格分层

### 1. 区域分层表

| 区域层级 | 适用地区 | 定价策略 | 币种建议 |
|----------|----------|----------|----------|
| **A区：高收入核心区** | 北美、西欧、北欧 | 对标国际标准价，价格最高 | 美元/欧元 |
| **B区：成熟市场区域** | 日本、韩国、澳大利亚、新西兰、中东高收入国家、新加坡及港澳台地区 | 保持成熟市场友好价 | 美元或当地币种 |
| **C区：中国大陆市场** | 中国大陆 | 独立人民币定价 | 人民币 |
| **D区：重点发展市场** | 除新加坡外的东南亚国家、东欧、拉美 | 区域友好价，促进市场拓展 | 美元或当地币种 |
| **E区：教育特惠市场** | 非洲、南亚、中亚及部分低收入地区 | 承担教育普惠和市场培育功能 | 美元或当地币种 |

这套分地区体系既能保持国际市场的商业价值，又能照顾东南亚、非洲、南亚等国际中文教育重点发展地区。以上区域划分仅用于商业定价和支付管理，不涉及政治或行政区划判断。

### 2. 区域适用原则

- **个人用户**：按账号注册地、支付地区、账单地址及实际使用地区综合判断。
- **机构用户**：按机构注册地、服务所在地和采购主体所在地综合判断。
- **学校、高校及其他教育组织**：按机构所在地、采购主体所在地及实际服务地区综合判断。
- **多校区项目或区域性项目**：按主要服务区域和合同主体综合核定。
- **E区特惠价格**：主要面向教育支持、公益合作、市场培育和发展型中文教育项目，实行人工审核。
- **价格体系**：可根据国家、地区、采购规模和服务内容进行弹性调整。
- **跨区使用**：对跨区使用、批量采购或特殊项目，平台保留按实际服务地区重新核定价格的权利。
- **各地区价格**：为独立市场价格，不是按照实时汇率进行的简单折算。
- **最终价格**：以正式报价、合同或结算页面为准。

- 若因资源规模、服务内容或产品结构调整导致 Plus 会员价格发生变化，已购买会员在当前有效期内不受影响；会员到期后，续费或重新购买时，按照届时平台公布的最新价格和权益规则执行。
- "中文+"专业资源、教材配套资源、教师教学资源和专项课件库，作为**"专业增值模块"**单独定价。

## 五、推荐、渠道与区域合作政策

平台可设置推荐与渠道合作机制，用于支持普通用户推荐、内容推广、机构线索推荐、区域合作及 Enterprise / 定制项目合作。相关奖励比例、结算方式和服务责任，应按照平台规则或双方合作协议执行。

### 1. 奖励计算基础

推荐与渠道合作奖励以**平台实际到账收入**为计算基础。平台实际到账收入，是指用户或机构完成有效支付后，扣除退款、税费、支付通道费、应用商店分成、优惠券抵扣及平台明确列明的其他成本后，平台实际获得的收入。

### 2. 推荐与合作类型

| 合作类型 | 适用对象 | 奖励比例 |
|----------|----------|----------|
| 普通用户推荐 | 普通注册用户通过个人专属链接或二维码推荐新用户购买 | 平台实际到账收入的 10% |
| 内容推广合作 | 内容创作者、教育博主、课程主理人、社群运营者 | 平台实际到账收入的 15% |
| 机构线索推荐 | 推荐学校、机构、高校项目、教育组织等有效机构客户线索 | 平台实际到账收入的 10% |
| 机构协助成交 | 推荐方参与客户沟通、试用推进、报价沟通并协助促成成交 | 平台实际到账收入的 15% |
| 区域合作方 | 承担本地市场推广、客户沟通、培训支持和基础服务 | 平台实际到账收入的 20% |
| Enterprise / 定制项目线索 | 推荐大型机构、区域项目、国家项目或定制项目线索 | 项目平台实际到账收入的 5% |
| Enterprise / 定制项目协助成交 | 推荐方深度参与项目沟通、方案推进、客户协调和成交支持 | 项目平台实际到账收入的 10% |

### 3. 奖励使用与结算

- 推荐奖励可用于抵扣推荐人本人后续订阅、续费或购买平台其他产品。平台也可在条件成熟后开放提现功能；提现应设置最低门槛，建议累计可提现金额不低于人民币 1000 元或等值外币，并按照平台财务、税务及身份核验规则执行。
- 推荐与渠道合作奖励不实时结算。平台可根据订单确认、退款期、客户回款、合同履行和服务交付情况，按月、季度或项目节点进行结算。
- 如涉及现金结算，推荐人或合作方应按照平台要求提供必要的身份、主体、收款、发票、税务及合规信息。
- 平台不设置多级返佣、团队返利、下线收益或以发展人数为核心的奖励机制。

## 六、税费、支付与退款说明

- **个人订阅**：是否包含当地适用税费，以最终结算页面为准。
- **机构版和 Enterprise / 定制项目报价**：除非另有说明，原则上为**未税价格**。
- **第三方平台购买**：通过苹果 App Store、Google Play 或其他第三方平台购买时，价格可能因平台抽成、当地税费和汇率产生差异。
- **数字服务税费**：欧盟、英国、日本、美国及其他地区的数字服务税费按照当地法律及支付渠道规定执行。
- **机构、学校及其他教育组织采购**：可根据当地要求提供正式报价、合同和发票或收据。
- **退款政策**：按照购买渠道、当地消费者保护规定及平台服务协议执行。
- **定制开发服务**：已启动的定制开发、内容建设、私有化部署和培训服务原则上不适用个人订阅退款政策。
